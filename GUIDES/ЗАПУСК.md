# Запуск проекту локально

## Вимоги

- Python 3.13+
- Node.js 20+
- npm

---

## Швидкий запуск (автоматично)

```bash
./start-dev.sh
```

Відкриється:

- Frontend: http://localhost:3000
- Backend API: http://localhost:8000

---

## Ручний запуск

### 1. Перше встановлення

**Backend:**

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

**Frontend:**

```bash
cd frontend
npm install
cd ..
```

---

### 2. Запуск бекенду

```bash
source venv/bin/activate
python api_server.py
```

Backend запуститься на http://localhost:8000

---

### 3. Запуск фронтенду

В новому терміналі:

```bash
cd frontend
npm run dev
```

Frontend запуститься на http://localhost:3000

---

## База даних

- **Тип:** SQLite
- **Файл:** `ahrefs_data.db` (створюється автоматично)
- **Міграції:** Alembic (застосовуються автоматично при запуску)

---

## Корисні команди

```bash
# Лише бекенд
./start-backend.sh

# Перевірка статусу API
curl http://localhost:8000/health

# Білд фронтенду для продакшену
cd frontend
npm run build

# Перегляд логів
tail -f backend.log
```

---

## Порти

- **3000** - Frontend (Vite dev server)
- **8000** - Backend (FastAPI)

**Важливо:** Фронтенд автоматично визначає API URL на основі hostname.
